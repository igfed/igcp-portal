No Single Sign-on Target for User

Access Manager WebSEAL could not determine the sign-on information for the resource you have requested.


Explanation

The resource you have requested requires the Access Manager WebSEAL server to sign you on to another Web server. However, your account does not have any sign-on information for the target Web server.




Solution

Your Account Administrator should add the appropriate sign-on target to your account information.
